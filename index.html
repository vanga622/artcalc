Расчёт угломера

У = Ац − Атн + mil/2. Если У ≥ mil → −mil. Если У < 0 → +mil. Для текущего орудия mil = 60-00 (6000).

Поправка по дальности

ΔXΣ = XT + XH + XW. Знак ΔП зависит от орудия и траектории: для 2Б14/2Б11 и 2Б9 настильной — минус; для 2Б9 мортирной — плюс.

Поправка по направлению

ΔZΣ = ZW + Z. Знак Wz: слева→направо «-», справа→налево «+». Результат прибавляется к угломеру.

Поправка на разность высот

Для мортирной траектории: Пh = (hц − hо)/100 × ΔПh. Для 2Б9 мортирной: ΔП = −Пh; для 2Б9 настильной: Пh = ((hц−hо)/100 × ΔПh) + ((hц−hо)/(0.001×S)), затем ΔП = Пh.
